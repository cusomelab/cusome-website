개인정보처리방침

시행일: 2026년 7월 17일

1. 총칙

주식회사 폰이지(이하 "회사")는 이용자의 개인정보를 소중히 여기며, 관련 법령을 준수합니다. 본 방침은 회사가 운영하는 쿠썸 셀러학교 웹사이트(cusome.kr)와 데스크톱 응용프로그램 쿠썸 에이전트(쿠썸 숏폼 에이전트, 쿠썸 롱폼 에이전트를 포함하며, 이하 "앱")에 적용됩니다.

2. 수집하는 개인정보 항목

  • 웹사이트: 문의·상담 시 이용자가 직접 입력한 이메일 주소 등 최소한의 정보
  • 앱: 별도의 회원가입 없이 동작하며, 아래 3항의 구글(YouTube) 사용자 데이터 외에 개인정보를 수집하지 않습니다.

3. 구글(YouTube) 사용자 데이터의 이용

앱의 "유튜브 업로드" 기능은 YouTube API 서비스(YouTube API Services)를 사용합니다. 이 기능을 사용하는 이용자는 YouTube 서비스 약관 및 Google 개인정보처리방침의 적용도 함께 받습니다.

요청하는 권한과 용도

권한용도
youtube.upload이용자가 직접 요청한 경우에만, 이용자가 제작한 영상을 이용자 본인의 유튜브 채널에 업로드합니다.
youtube.readonly연결된 채널의 이름과 프로필 사진을 앱 화면에 표시하여, 어느 채널에 업로드되는지 확인할 수 있게 합니다.
🔒 저장 위치 — 구글 로그인으로 발급된 인증 토큰은 이용자 본인 PC의 앱 폴더에만 저장됩니다. 회사 서버로 전송·수집·보관하지 않으며, 회사는 이용자의 유튜브 계정이나 데이터에 접근할 수 없습니다.
  • 제3자 제공: 구글 사용자 데이터를 제3자에게 제공·판매·공유하지 않습니다.
  • 광고 목적 사용: 구글 사용자 데이터를 광고 목적으로 사용하지 않습니다.
  • 앱의 구글 사용자 데이터 취급은 Google API 서비스 사용자 데이터 정책(제한적 사용 요건 포함)을 준수합니다.

4. 데이터 보관 및 삭제

  • 인증 토큰은 앱에서 [연결 해제] 버튼을 누르면 즉시 삭제됩니다.
  • 이용자는 구글 계정 보안 설정에서 언제든지 앱의 접근 권한을 직접 회수할 수 있습니다.
  • 앱(폴더)을 삭제하면 PC에 저장된 관련 데이터가 모두 함께 삭제됩니다.
  • 웹사이트 문의 정보는 처리 완료 후 지체 없이 파기합니다.

5. 쿠키 및 로컬 저장소

웹사이트는 화면 테마 설정 저장 등 이용 편의를 위해 브라우저 로컬 저장소를 사용할 수 있으며, 해당 정보를 외부로 전송하지 않습니다.

6. 이용자의 권리

이용자는 언제든지 자신의 개인정보에 대한 열람·정정·삭제를 아래 문의처로 요청할 수 있으며, 회사는 지체 없이 조치합니다.
