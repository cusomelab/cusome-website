FAQ

자주 묻는 질문

더 궁금한 내용은 언제든 문의해 주세요.

쿠썸 에이전트는 웹 서비스인가요?＋

현재 영상 에이전트는 Windows PC에 설치해 로컬에서 실행하는 방식입니다. 브라우저에서 대시보드를 열지만 작업 엔진은 내 컴퓨터에서 동작합니다.

다른 프로그램이나 API 결제가 필요한가요?＋

영상 제작에는 Vrew, Google Flow 등 연결 서비스가 사용되며 AI 대본 자동 생성 시 선택한 AI 제공자의 API 키가 필요합니다. 각 서비스의 요금과 생성 한도는 별도입니다.

컴퓨터를 잘 몰라도 사용할 수 있나요?＋

네. 처음 한 번의 로그인과 설정부터 자동 제작까지 순서대로 설명한 한글 가이드를 제공합니다. 가장 쉬운 자동 제작 모드부터 시작할 수 있습니다.

YouTube 업로드도 지금 가능한가요?＋

현재는 완성 MP4 제작까지 제공하며, 채널 인증 후 제목·설명·태그·예약 공개를 포함한 YouTube 등록 기능은 추가 개발 중입니다.

작업이 중간에 멈추면 처음부터 해야 하나요?＋

아니요. 같은 이름과 대본으로 다시 실행하면 완료된 단계는 재사용하고 멈춘 지점부터 이어서 제작할 수 있습니다.
